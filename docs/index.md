# 18xx サマリーサイトへようこそ

このサイトは、18xxボードゲームのルール要点をスマホで読みやすく、印刷もできる形でまとめたサマリー集です。

## 特徴

- タブ形式：SR（株式ラウンド）/OR(運営ラウンド)/セットアップ・早見表で情報を整理
- モバイル対応：スマートフォンでゲーム中に確認しやすい
- 印刷可能：各ページの印刷ボタンで簡易紙出力、テキスト出力にも対応しているので気に入らなければ自分用のサマリーに編集も可能

## 使い方

1. 上部メニューの「Games」から目的のゲームを選択
2. SR/OR/セットアップのタブで必要な情報を確認

## フィードバック・追加リクエスト

誤記の修正や新しいゲームの追加リクエストは、[GitHub Issues](https://github.com/satamoto7/18xx-summary-jp/issues)までお願いします。
